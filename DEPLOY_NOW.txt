🎯 YOUR IMMEDIATE TODO LIST
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Get Discord Bot Credentials (5 min)
   □ Go to: https://discord.com/developers/applications
   □ Click "New Application" → Name: "Streamer X Bot" → Create
   □ Click "Bot" on left → "Add Bot"
   □ Under TOKEN section, click "Copy" → SAVE: DISCORD_BOT_TOKEN
   □ Go to OAUTH2 → URL Generator
     - Scope: ✓ bot
     - Permissions: ✓ Send Messages, Read Messages, Embed Links
     - Copy generated URL
   □ Open that URL → Select your server → Authorize
   □ Right-click your Discord server → Copy Server ID → SAVE: DISCORD_GUILD_ID
   □ Get CLIENT ID from application settings → SAVE: DISCORD_CLIENT_ID

   YOU NOW HAVE 3 THINGS:
   - DISCORD_BOT_TOKEN
   - DISCORD_CLIENT_ID
   - DISCORD_GUILD_ID


STEP 2A: Deploy to Railway (5 min) - RECOMMENDED ⭐
   □ Go to: https://railway.app
   □ Click "Login" → "Login with GitHub"
   □ You'll be guided to create GitHub repo automatically
   □ Click "Create new Project" → "Deploy from GitHub repo"
   □ Select "streamer-x-bot" repository
   □ Railway detects it's Node.js, auto-configures
   □ Go to "Variables" tab
   □ Add these 3 variables:
      DISCORD_BOT_TOKEN = [paste your token]
      DISCORD_CLIENT_ID = [paste your client ID]
      DISCORD_GUILD_ID = [paste your guild ID]
   □ Save
   □ Railway auto-deploys! 🎉

   BOT IS NOW LIVE 24/7!


STEP 2B: Deploy to Replit (if you prefer) - EASIEST ⭐
   □ Go to: https://replit.com
   □ Click "Create Repl" → "Import from GitHub"
   □ Paste: https://github.com/YOUR_USERNAME/streamer-x-bot
   □ Create
   □ Click "Secrets" (lock icon)
   □ Add your 3 variables:
      DISCORD_BOT_TOKEN
      DISCORD_CLIENT_ID
      DISCORD_GUILD_ID
   □ Click "Run"
   □ Bot starts!

   TO KEEP RUNNING 24/7:
   □ Click "Upgrade" (shows pricing)
   □ Or use: https://uptimerobot.com (monitors and restarts)


STEP 3: Verify Bot is Online (1 min)
   □ Go to your Discord server
   □ Type: /genkey username:test
   □ If command appears → BOT IS WORKING! 🎉
   □ Generate a key to test functionality

LOCAL TESTING (Before Deployment - Optional)
═══════════════════════════════════════════════════════════════════════════════

Want to test locally first? Do this:

1. Create .env file in project folder (same as .env.example but with REAL values):
   
   DISCORD_BOT_TOKEN=YOUR_REAL_TOKEN_HERE
   DISCORD_CLIENT_ID=YOUR_REAL_ID_HERE
   DISCORD_GUILD_ID=YOUR_REAL_GUILD_ID_HERE

2. In PowerShell, in project folder:
   node discord-bot.js

3. Look for: "Bot is online!" message

4. Test in Discord:
   /genkey username:test

5. If it works → Ready for cloud deployment!

6. Stop bot: Ctrl+C
