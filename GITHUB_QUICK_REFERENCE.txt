🚀 3 COMMANDS TO PUSH TO GITHUB
═══════════════════════════════════════════════════════════════════════════════

1. In PowerShell, navigate to your project:
   cd "c:\Users\hp\Desktop\New folder"

2. Initialize Git and add files:
   git init
   git add .
   git commit -m "Initial commit: Discord bot ready for deployment"

3. Push to GitHub:
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/streamer-x-bot.git
   git push -u origin main

(Replace YOUR_USERNAME with your GitHub username)
(When asked for password, use GitHub Personal Access Token, not password)

🔐 SECURITY CHECK BEFORE PUSHING
═══════════════════════════════════════════════════════════════════════════════

Before you push, open these files and verify:

.env.example → Should look like:
   DISCORD_BOT_TOKEN=YOUR_BOT_TOKEN_HERE ← Placeholder, not real!
   DISCORD_CLIENT_ID=YOUR_CLIENT_ID_HERE
   DISCORD_GUILD_ID=YOUR_GUILD_ID_HERE
   ✓ Contains NO real tokens

discord-bot.js → Should load tokens from .env:
   require('dotenv').config();
   const TOKEN = process.env.DISCORD_BOT_TOKEN;
   ✓ Reads from .env (not hardcoded)

package.json → Should have dependencies:
   "dependencies": {
     "discord.js": "^14.13.0",
     "dotenv": "^16.3.1"
   }
   ✓ No secrets here

If all checks pass → Safe to push! ✓

📋 STEP-BY-STEP COMMANDS (Copy & Paste)
═══════════════════════════════════════════════════════════════════════════════

1. Open PowerShell and navigate:
   cd "c:\Users\hp\Desktop\New folder"

2. Configure Git (first time only):
   git config --global user.name "Your Name"
   git config --global user.email "your@email.com"

3. Check what will be committed:
   git status

4. Add all files (respects .gitignore):
   git add .

5. Commit:
   git commit -m "Initial commit: Discord bot ready for Railway deployment"

6. Set up remote (replace username):
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/streamer-x-bot.git

7. Push to GitHub:
   git push -u origin main

8. When asked for password:
   → Go to: https://github.com/settings/tokens
   → Create Personal Access Token
   → Paste as password
